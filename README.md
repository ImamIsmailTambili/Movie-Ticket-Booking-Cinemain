# 🎬 Cinemain – Movie Ticket Booking Web App

Cinemain is a full-stack web application for booking movie tickets. This project consists of a frontend user website, admin panel, and backend API for managing movie data and transactions.

---

## 🚀 Features

### 🎥 User Features
- Browse available movies  
- View movie details  
- Responsive UI for desktop & mobile  
- Interactive movie slider  

### 🛠️ Admin Features
- Manage movie data (CRUD)  
- Update movie listings  
- Backend integration  

### ⚙️ Backend Features
- RESTful API for movie data  
- Database integration  
- Data handling using ORM  

---

## 🏗️ Tech Stack

### 🎨 Frontend
- Next.js (App Router)  
- TypeScript  
- Tailwind CSS  

### ⚙️ Backend
- Node.js  
- Express.js  
- Prisma ORM  

### 🗄️ Database
- PostgreSQL
